== Description ==

WMD_Admin is a Worpdress plugin which makes [WMD](http://wmd-editor.com/) the default posts editor in the admin interface.

== Installation ==

1. Put wmd-admin into your /wp-content/plugins/ directory.
2. Go to your profile page and check "Disable enhanced editor".
3. Be sure to install a Markdown parser (such as
   <http://wordpress.org/extend/plugins/markdown-for-wordpress-and-bbpress/>).
4. Activate wmd-admin in your plugins page.
